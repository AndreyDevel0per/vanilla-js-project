# Vanilla js project

## Скрипты

**Запуск dev-режима:**
`npm start`

**Запуск build-режима:**
`npm run build`

## Правила работы с git

1. Главная ветка - main;
2. Ветка для разработки - dev;
3. Ветвиться от ветки dev. После окончания работы сделать MR в dev;
4. Ветки для работы называть "task: short description"

## Правила для коммитов

Использовать формат [conventional commits](https://www.conventionalcommits.org/en/v1.0.0/)

## Стек технологий

1. JavaScript;
2. Стили: SASS;
3. Внешние библиотеки: None;
4. Линтирование: ESLint, Stylelint;
5. Архитектура: Модульная;
6. Сборщик: Vite.
